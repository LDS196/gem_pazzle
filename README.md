:seedling: <a href="https://rolling-scopes-school.github.io/lds196-JSFE2022Q3/codejam/" target="_blank">Gem Pazzle</a>
---
это приложение-игра в пятнашки. Пользователь должен выстроить пазлы по порядку за наименьшее количество ходов. Так же пользователь может выбрать размер игрового поля(количество пазлов), включить/отключить звуковое сопровождение. Функция сохранения игры недоступна.
